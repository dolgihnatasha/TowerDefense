Tower Defense!

To run requires python3.4 and PyQt4

===Map Editor===:
    Что бы добавить дорогу кликайте по клеткам на поле. Дорога
    должна начинаться слева и заканчиваться справа

    buttons*:
        s - сохранить карту
        m - смена режима: добавить/удалить клетки
    * При проблемах с запуском смените язык по умолчанию на английский

===Game===:
Выберите башню и поставте на карту. Зажмите Shift,
чтобы выбор башни не сбрасывался
